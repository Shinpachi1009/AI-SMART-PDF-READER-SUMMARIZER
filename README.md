# PDF RAG Summarization Application

A Node.js application that uses RAG (Retrieval-Augmented Generation) to summarize PDF documents with an intuitive web interface.

## Prerequisites

1. **Node.js** (v14 or higher)
2. **Python** (3.8 or higher)
3. **Conda** (recommended for Python environment management)

### 1. Installation
pip install -r requirements.txt

### 2. Set up Python environment and dependencies
cd python
conda env create -f environment.yml
conda activate ragpdf

### 3. Run the application
cd server
npm install <!--if not yet installed-->
npm start
